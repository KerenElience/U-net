# U-Net
复现[U-Net论文](https://arxiv.org/html/1505.04597v1#S1)，强化记忆

## 具体细节
- 为方便进行多分类的segment任务将输出层改为可选，将sigmoid改为softmax
- 原文中的下采样方法从图示标注上来看是maxpool，替换为3X3的Conv2d，步长设置为2，padding为1。相对于最大池化，使用卷积进行下采样能提取更多的细节
- 原文中的卷积核采用reflect方式进行padding的，使得一个感受野获取更多的信息
- 架构上面采用了Encoder-Decoder模式
- 作为2015年的文章，在架构上采用了类似Resnet的残差连接方式（上采样中的拼接）
- 转置卷积是一个可学习的权重，能在网络训练过程自适应找到最佳的采样方式，代价是训练和推理的速度不如插值法
- 转置卷积核使用2x2，能够避免伪影，栅格出现
- Unet在少量的样本训练上就能取得不错的效果，缺点是相对于Yolo，VGG，Resnet等网络数据的标注很痛苦，但是相对于Yolo框架来说能够很好的计算识别区域的面积方便后续的精确量化
